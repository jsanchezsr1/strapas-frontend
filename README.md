# Strapas — Enhanced AI App Builder Frontend

A premium Next.js + Tailwind CSS front-end starter inspired by the polished style of modern AI app builder websites.

## Run locally

```bash
pnpm install
pnpm run dev
```

## Included

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Premium landing page sections
- Hero mockup, deployment, showcase, pricing, FAQ, CTA

## Notes

This is an original front-end starter, not a clone of any specific brand. It is designed to capture a strong modern AI startup feel.
